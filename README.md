## Cinemash


## General info
Cinemash is web application for browsing and managing favroties movies.

## Technologies
Project is created with:
* React.js
* Firebase (Firestore DB)

	
## Setup
To run this project, install it locally using npm:

```
$ cd Cinemash
$ npm install
$ npm start
```
